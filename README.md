# CSight
Merupakan aplikasi yang diharapkan dapat membantu Orang dengan gangguan pengelihatan dalam mengetahui Object maupun benda - benda yang ada di sekitarnya dengan demikian diharapkan dapat membantu dalam melakukan transaksi menggunakan uang, berjalan dan lain sebagainya  fitur Object Detection yang dikembangkan melalui metode Sum Of Absolute Difference kemudian input yang digunakan berupa citra yang ditangkap secara realtime menggunakan kamera Device Android , hasil deteksi object tersebut akan di output kan berupa suara
sebagai contoh :
Saat sistem mendeteksi object meja melalui kamera android, maka sistem akan memberikan output berupa suara dengan berkata "Di depan ada meja"
